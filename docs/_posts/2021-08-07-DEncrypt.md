## Method

#### Md5
md5
```
string s="123456";
DEncrypt.Md5(s)
DEncrypt.Md5(s,true)
```
output:
```
e10adc3949ba59abbe56e057f20f883e
E10ADC3949BA59ABBE56E057F20F883E
```

#### Sha1
sha1
```
DEncrypt.Sha1(s)
DEncrypt.Sha1(s,true)
```
output:
```
7c4a8d09ca3762af61e59520943dc26494f8941b
7C4A8D09CA3762AF61E59520943DC26494F8941B
```

#### Encrypt
Encrypt data.
```
DEncrypt.Encrypt(s)
DEncrypt.Encrypt(s,"luohongwei")
```
output:
```
BE3017EA5B9C68C6
D8E3D5B74BEA83D2
```


#### Decrypt
Decrypt data.
```
DEncrypt.Decrypt("BE3017EA5B9C68C6")
DEncrypt.Decrypt("D8E3D5B74BEA83D2","luohongwei")
```
output:
```
123456
123456
```
